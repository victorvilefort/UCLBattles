<h1>UCLBattles</h1> <br>
<p>Aqui será o repositório do projeto individual, o objetivo é a construção de um site onde o usuário pode criar uma conta e um jogador virtual para comparar seus atributos/stats com as maiores referências do futebol mundial atualmente, especificamente aqueles que disputam a tão sonhada <strong>UEFA Champions League</strong o maior campeonato entre clubes do planeta.
Para a construção desse site será necessário utilizar:</p>
<li>HTML</li> 
<li>CSS</li> 
<li>JavaScript</li>
<li>Node.js</li>   
<li>Chart.js</li>  
<li>MySQL</li> 
<br>
 <p> O banco de dados (MySQL) será o responsável por armazenar todos os dados referente ao Login/Cadastro/Jogador-Virtual por meio de tabelas como: player-user, exceto os jogadores de comparção que possuem seus dados 'mockados' no próprio site, que serão consumidos para que aconteça a comparação entre stats do jogador selecionado e do jogador virtual criado pelo usuário.
<br>
<br>
Esse projeto é essencialmente voltado para aplicação de conhecimentos adquiridos na faculdade (São Paulo Tech School) até o presente momento.
</p>
